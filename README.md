# Win32 Hook Demo
reveal and detect of common hooks under win32

## wat do
- reveal IAT, and EAT hook.

- reveal software breakpoint, hard breakpoint.

- detects inline, IAT, and EAT hook.

- detects software breakpoint, hard breakpoint.
